# Supported Content

This document defines the currently supported slices of content that are officially maintained and prioritized.

## Current Supported Slice
The official supported slice follows a strict milestone-based approach.

### 1. KotOR I: Endar Spire (Milestone A)
Primary focus. Must be **fully completable** with **no crashes** and **no softlocks**.

### 2. KotOR I: Taris Entry Slice (Milestone B)
Secondary focus, prioritized only after Milestone A is secured. Must be **fully completable** with **no crashes** and **no script halts**.

## Scope Management
- **No Scope Creep:** We are not optimizing for all BioWare games at once. Content outside the defined supported slice will not receive priority until the current slice hits its strict playability definition.
- **Feature Freeze:** Nonessential features will NOT be added during the stabilization of the current supported slice.

## Issue Tracking Categories
Every issue mapped to the supported parameters must fall into one of the following tags to be properly triaged:
- `crash`
- `progression blocker`
- `logic bug`
- `visual bug`
- `missing parity`
- `qol`
